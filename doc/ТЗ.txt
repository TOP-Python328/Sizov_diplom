Виртуальные лаборатории 2.0



    Описание:

Веб приложение, которое является интерактивным задачником и инструментом для проверки знаний по основам информатики для учеников начальной и средней школы.
Учитель может назначать задания ученикам и проверять их решение. Ученики выполняют задания назначенные учителем или тренируются внутри симулятора/тренажера.


    Ролевая модель:

Администратор/группа Администраторы - полный доступ к приложению, управляет всеми пользователями и просматривает все страницы;
Учитель/группа Учитель - управление пользователями группы Ученик, просмотр страниц/сайтов предназначенных для учителя;
Ученик/группа Ученики - минимальные права доступа, просмотр страниц/сайтов предназначенных для ученика. Работа в тренажере.

Администратор создается вручную. Он может управлять пользователями через приложение. Для регистрации клиента с ролью "Учитель" заполняется форма. 
Учитель сам регистрирует учеников заполняя соответствующую форму и передает данные для входа ученикам.

    Интерфейс:

Графический интерфейс пользователя (GUI)



    Функциональность:

Все задания подразделяются на категории - "Лаборатории": Переливания, Перекладывания, Переправы, Взвешывания ... Для каждой категории есть свой тренажер. 
Для ознакомления с процессом решения задачи в выбранной лаборатории на форме расположена кнопка запуска демонстрации решения задачи. 
Демонстрационное решение можно посмотреть поэтапно – воспользовавшись кнопками. Во время демонстрации ученику поэтапно показываются все действия, которые нужно
выполнить в лаборатории для решения данной задачи. 

Учитель выдает задание, с меткой времени и временем на выполнение. Ученки должны выполнить это задание на своей странице/личный кабинет.

Для анализа работы класса и создания рейтингов учащихся учитель может сформировать статистику, выбрав класс, ученика и задачи, по которым будет выполнен 
сбор статистики – т.е. набранных баллов. 

Ученики объединяются в классы. Учитель может назначить задачи коллективно, всем ученикам класса, так и индивидуально.

Должна быть возможность перевода учеников из одного класса в другой, продумать над автоматическим переводом после оканчания учедного года с возможностью ручного редактирования.

В приложении присутствует тренажер, с помощью которого пользователи могут тренироваться.
  


    Дополнительный функционал:

Чат для общения и обсуждения домашнего задания.
Справочная информация и литература.
